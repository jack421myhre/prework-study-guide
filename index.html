Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element contains all visible content shown to the user.

CSS

  • Margin indicates the space around the outside of an element.
  • Padding indicates the space around the content inside of an element.

Git

  • git status: checks what branch you are currently on.
  • git checkout -b "branch-name": creates a new branch and switches to it.
  • git add -A: adds all current changes from the branch to the staging area.
  • git commit -m "description": creates a commit with a description of the changes made. Updates the branch with the newly committed code.
  • git pull origin main: "pulls" a branch's changes to the local environment. Origin main indicates where it is being pulled from, in this case, the main branch. Also syncs the remote base branch with the local branch.
  • git push origin "branch-name": pushes the local changes to the remote GitHub branch.
